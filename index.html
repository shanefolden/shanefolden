PROJECTS

Blackmailer.xyz

Blackmailer.xyz is a website designed for people who struggle with motivation to complete tasks. It allows users to set a deadline for themself, and upload an embarrassing picture of themself along with the email address of a person they know.

If they fail to submit proof of completing the task, the picture gets sent out to the person, effectively allowing the user to blackmail themself into completing their task.

Website: blackmailer.xyz
Source Code: Github repo

Lights Message Board

The Lights Board is a project I made for my friends to send me messages. They submit a message through a form on the site, and it lights up on my wall.

It works by having a python script that scrapes my email for form submissions. It then sends the message over serial to an arduino that lights up the individual LEDs. For more info, check out the README in the source code.

Source Code: Github Repo

Fridaybot

Fridaybot is a reddit bot I made. It parses comments made by users on reddit, and detects when a user asks "Is it friday yet?". It then responds with whether or not it is Friday, and encourages the user to finish the week.

It was made in Python using Praw, Reddit's API wrapper. It has been active for 1.5 years, and in that time has answered over 100 comments, and accumulated over 4500 karma (Reddit's 'like' system).

Source Code: Github Repo
